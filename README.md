# HeraPorto Static Public Portfolio

Versi statis dari halaman publik project HeraPorto.

Project Laravel/Inertia aslinya sudah dikonversi menjadi file statis:

- `index.html`
- `projects.html`
- `tutorials.html`
- `project-detail.html`
- `tutorial-detail.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `assets/logo.svg`

Data profil, tech stack, project, tutorial, dan social link berada di:

```text
assets/js/app.js
```

## Preview Lokal

Buka langsung:

```text
index.html
```

## Deploy

Folder ini bisa diupload ke GitHub Pages, Netlify, Vercel, atau hosting statis biasa.

Untuk GitHub Pages, push folder ini ke repository, lalu aktifkan:

```text
Settings > Pages > Deploy from a branch > main > /root
```

## Catatan

Ini hanya halaman publik. Fitur Laravel seperti admin CMS, database, login, upload gambar, CKEditor, dan pengiriman email tidak ikut dibawa.
